## 各模块功能

### base

提供一个 session ，并提供 login 方法调用，建议每个子功能类都继承 base 类进行操作。

### ping-jiao

实现评教功能，代码来源于大神 @linjinjin123(https://github.com/linjinjin123) 的一个一键评教脚本，但是因为现在已经不能用了，所以个人作了一些微小的改动。

usage: python ping-jiao.py
